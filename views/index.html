Exercise tracker

Create a New User

POST /api/exercise/new-user

Add exercises

POST /api/exercise/add

GET users's exercise log: GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

Example URL: https://fccextrack.glitch.me/api/exercise/log?userId=userid&from=yyyy-mm-dd&to=yyyy-mm-dd&limit=a valid number
where userid is from Create a New User form above and from and to are valid dates entered by user in YYYY-mm-dd format in the URL
